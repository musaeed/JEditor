Follow the simple steps to update your JEditor

  1. The updates are NOT provided by the Ubuntu Software Center.
  2. Please connect your pc to internet before proceeding.
  3. The updator will ask you for your computer password to update. This is required to update the binaries of JEdtor as they are placed in the root folder of your system.
  4. After the update is complete, please restart JEditor to see the effects.

To update the JEdtor manually, first uninstall the current JEdtor (you can use Synaptic Package Manager for that) and then install the latest debian package which can be downloaded from here.
